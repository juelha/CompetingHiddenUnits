# CompetingHiddenUnits
Implementation of "Unsupervised learning by competing hidden units" by Krotov and Hopfield (2019)



## Performance:

paper on mnist biological network for  2,000 hidden units: 
    - training error = 0.40% 
    - test error = 1.46%
